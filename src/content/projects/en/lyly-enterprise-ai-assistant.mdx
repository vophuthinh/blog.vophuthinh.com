Lyly was designed to help HPT executives make faster, data-driven decisions from fragmented internal knowledge.

Core capabilities:

- Real-time text and voice conversations via WebSocket.
- Hybrid RAG (vector + keyword) with corrective retrieval.
- Multi-step reasoning for complex queries.
- Personalized memory for context continuity.

Tech architecture:

- FastAPI backend and Next.js frontend.
- Gemini-powered reasoning and Qdrant vector database.

Outcome:

- Won **2nd Prize** in HPT's "AI in Business" competition.
- Delivered a production-grade enterprise AI assistant used for decision support.
