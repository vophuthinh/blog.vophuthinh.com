This project combines a full multi-vendor commerce stack with AI-powered sentiment classification.

What it delivers:

- Automatic positive/negative review classification using BERTweet.
- Real-time seller-buyer communication via Socket.io.
- Seller visibility into sentiment trends for product/service improvement.
- Complete e-commerce capabilities with payment and media integrations.

Result:

- Turned unstructured review text into practical decision signals for sellers.
