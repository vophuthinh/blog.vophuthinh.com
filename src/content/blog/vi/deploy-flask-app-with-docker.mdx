Docker giúp chuẩn hóa môi trường deploy và giảm khác biệt giữa máy dev với production.

## Dockerfile tham khảo

```dockerfile
FROM python:3.12-slim

WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .
EXPOSE 8000
CMD ["gunicorn", "app:app", "--bind", "0.0.0.0:8000"]
```

## Lưu ý khi deploy

- Dùng image nhỏ để giảm thời gian build/pull.
- Không chạy container bằng user root.
- Đặt healthcheck để hệ thống tự phát hiện lỗi.

Kết hợp Docker với CI/CD sẽ giúp vòng lặp release nhanh và ổn định hơn rất nhiều.
